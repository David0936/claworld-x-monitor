- 🔓 默认免登录、开箱即用；登录密码改为「设置 → 登录保护」里的可选开关（商用/公网部署可自设密码，信息流仍公开、仅「设置」需登录）
- 📖 新增「使用教程」栏目：在哪拿 X API、怎么设博主、怎么配飞书、怎么配 AI，四步图文说明
- ⬇️ 新增「一键更新」：检测到 GitHub 新 release 时，设置页弹出更新按钮，点一下自动拉最新代码并重启；`config.json`/`data` 已忽略，配置原样保留
- 🐛 修复设置/关于页加载慢：GitHub 版本检查失败时也缓存 1 小时、超时降到 3 秒，不再每次打开都阻塞重试
